## 2. Typography

*   **Content / Headings Font**: `Inter`, `DM Sans`, sans-serif
*   **Data / Code Font**: `JetBrains Mono`, monospace
*   **Scale**:
    *   `h1`: `1.75rem` (Bold, tracking-tight)
    *   `h2`: `1.25rem` (Semi-bold, tracking-tight)
    *   `h3` / Labels: `0.875rem` (Medium)
    *   Body text: `0.875rem` (Regular)
    *   Small / Muted text: `0.75rem` (Regular)

---

## 3. Component Design Rules

### Theme Toggle
*   A header Sun/Moon button toggles the `.dark` class on the `<html>` or `<body>` element.
*   All visual elements transition color schemes smoothly using `transition: background-color 0.3s ease, border-color 0.3s ease, color 0.2s ease`.

### Sidebar Navigation
*   Fixed vertical layout on the left side.
*   Includes minimal chrome, prominent branding, navigation links with left alignment, custom icons, and interactive active/hover states.

### Card Panels
*   Cards use `border: 1px solid var(--border)` and `border-radius: 12px`.
*   Include generous padding (`1.25rem` / `20px`) inside cards and consistent gaps.
*   Background utilizes subtle backdrop blurring for a high-quality glassmorphic feel.

### Agent Execution Tree
*   Nodes are interconnected using vertical lines.
*   Active steps use `--accent` with a double-ring glowing animation.
*   Success steps show CheckCircle in `--success`, and failed runs show XCircle in `--error`.

### Split-Pane Workflow Terminal
*   **Left Console**: Monospaced terminal window with simulated console headers. Renders raw Python subprocess logs.
*   **Right Insights Viewer**: Streams real-time formatted agent thoughts and writer reports with elegant syntax highlighting for generated code blocks.
